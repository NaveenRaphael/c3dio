A PC floating point formatted C3D file that contains multiple custom GROUPS and PARAMETERS in
addition to the standard C3D parameter values.

02/26/2009  08:35 PM         2,426,368 Vicon_analysis.c3d

The custom GROUPS are SUBJECTS, EVENT, EVENT_CONTEXT, ANALYSIS, and EMG - the first four were
created by Vicon to describe the trial data, the EMG group was created by Motion Lab Systems to
store their EMG analysis configuration.
